# 🏗️ BOQ Sync

เครื่องมือแปลง **BOQ เก่า (ฟอร์มคุณกี้)** → **Master Template BOQ** พร้อม export เป็น Excel

## ใช้งานออนไลน์
👉 เปิดที่: **https://mahantapon.github.io/boq-sync/**

## วิธีใช้
1. ตั้งค่า Job Code / Group Code / Group Name
2. ลากไฟล์ BOQ เก่า (.xlsx) มาวาง — ระบบเลือกชีท "BOQ ต้นทุน" ให้อัตโนมัติ
3. ตรวจสอบตาราง Sync Check
   - 🩷 สีชมพู = ต้นทุน → คอลัมน์ (Budget)
   - 🩵 สีฟ้า = ราคาขาย → คอลัมน์ (BOQ)
4. กด **Export** → ได้ไฟล์ Excel ตามฟอร์ม Master Template พร้อมสูตรครบ

## จุดที่ระบบจัดการให้อัตโนมัติ
- **แถวแจกแจงรายละเอียด** ที่ไฟล์เดิมไม่นับในยอดรวม → ไม่นับตาม (มี checkbox เปิดได้)
- **แถวเหมารวม** (เช่น "คงเหลือ") ที่ไม่มี QTY → นับเป็น QTY 1 ไม่ให้ต้นทุนตกหล่น
- **แถวที่ราคาต่อหน่วยเดิมไม่ตรงยอดรวม** → ยึด ยอดรวม ÷ QTY

## ความปลอดภัย
ประมวลผลในเบราว์เซอร์ทั้งหมด ข้อมูล BOQ **ไม่ถูกอัปโหลด**ไปที่ไหน
(ต้องต่ออินเทอร์เน็ตครั้งแรกเพื่อโหลดไลบรารีอ่าน/เขียน Excel)

## หมายเหตุ
คอลัมน์ Mat. Code / Cost Code / Unit Code เป็น master data ของ ERP
ที่ไม่มีในไฟล์เก่า — ต้องเติมเองใน Excel หลัง export
